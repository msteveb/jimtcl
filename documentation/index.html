Jim Tcl Documentation

Overview

Jim includes an extensive User Reference Manual

Jim is highly compatible implementation of the Tcl programming language. Despite some differences, it is not a new language, so much of the available Tcl documentation is perfectly suited to learning about Jim Tcl.

To know more about Tcl, you can look at the great Tcler's Wiki Site.

Do you need a Tcl tutorial? Salvatore Sanfilippo wrote one that you can read On Line.
