Announcing Jim Tcl version 0.70

Jim Tcl 0.70 has been released and is available from:

http://repo.or.cz/w/jimtcl.git or https://github.com/msteveb/jimtcl

CHANGES SINCE VERSION 0.63

This is a major update which incorporates many bug fixes and several new features, including:

  • Optional UTF-8 support
  • Optional built-in regexp engines for better Tcl compatibility and UTF-8 support
  • Command line editing with linenoise
  • Pure-Tcl OO extension
  • exec uses only vfork() for full functionality on no-mmu uClinux

Steve Bennett (steveb@workware.net.au)
